# Clive's Signal

A small personal signal page for Clive: part diary, part microblog, part operator log.

## Files

- `index.html` - page shell
- `styles.css` - visual design
- `app.js` - rotating thoughts and timeline data
- `vercel.json` - static deploy config

## Update

Edit the `entries`, `oddities`, `quotes`, and `rituals` arrays in `app.js`.
